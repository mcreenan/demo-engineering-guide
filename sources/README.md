# Local source material

The demo ships three sample code repos (`code/platform`, `code/web`, `code/infra`),
an engineering wiki (`docs/wiki`), and a help center (`docs/help-center`) so
`pal outline` and `pal draft` have real, mutually-consistent material to read. In
your own project, clone your repos into `code/<name>` and docs into `docs/<name>`.
These directories are gitignored (tracked via `.gitkeep`).
